Rooky

A minichess AI written in Ruby.


Rooky is a minichess playing AI developed for Bart Massey's Advanced Artificial Intellegence course. It is built primarily around the negamax algorithm for zero-sum, two player games. The code base is written entirely in Ruby, including Rspec test coverege.
