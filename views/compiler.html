MiniJava Compiler

A MiniJava compiler written in Java.


For Portland State's compiler course, students are required to write a complete compiler for the MiniJava language, a subset of Java. First term objectives focus around creating the front-end of the compiler, including lexical analysis, parsing, and static analysis / type checking.

    Due to the fact that this is coursework that doesn't change much term to term, the source is closed-source, in a private repository.
