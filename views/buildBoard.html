BuildBoard

A web data visualization tool written primarily in Ruby and Javascript.


The Build Board is a web app created as an intern project for Puppet Labs in order to visualize key metrics related to the company software release process. It is built on top of the Sinatra web framework, and heavily utilizes the D3 Javascript library to create real time graphs of various packaging statistics.
